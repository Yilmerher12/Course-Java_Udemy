# 🚀 60 Days of Java : The Complete Java Masterclass


Este repositorio contiene mi progreso, ejercicios y proyectos desarrollados durante el curso de **Java** de "60 Days of Java : The Complete Java Masterclass" Denis Panjuta. Mi objetivo es documentar mi proceso de aprendizaje, fortalecer mis bases en lógica de programación y tener un respaldo de mis prácticas diarias.

---
### 🌐 Por que en ingles?
He decidido llevar este repositorio y mi código en inglés porque:
* **Global Standard:** Es el idioma universal de la tecnología. Aprender a pensar y nombrar variables en inglés es un paso obligatorio para un desarrollador profesional.
* **Skill Sharpening:** Como estudiante de ADSO, busco mejorar mi fluidez técnica. Documentar en inglés me obliga a practicar el vocabulario de la industria diariamente.
* **Better Opportunities:** Me prepara para trabajar en entornos internacionales y acceder a la mejor documentación técnica disponible.
